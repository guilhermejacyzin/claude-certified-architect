# Trilha Academy Expandida em PT-BR

Esta pasta organiza os primeiros conteúdos visíveis no learning path da
Anthropic Academy em uma versão textual aprofundada em português.

Ela não copia as aulas oficiais. Em vez disso, usa os títulos e temas do
currículo como índice e desenvolve explicações próprias, exemplos, checklists e
atividades para estudo.

## Critério

Cada seção deve responder quatro perguntas:

1. O que é?
2. Quando usar?
3. Como implementar?
4. Qual erro a certificação provavelmente cobraria?
